Short Description: A plugin that adds a wishlist feature for your website, allowing users to save and manage their favorite items.


== Description ==

Wishlist Everywhere allows users to add any post type (products, blog posts, events, etc.) to a personal wishlist. It works seamlessly with WooCommerce and custom post types, giving users a flexible way to save their favorites.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wishlist-everywhere` directory.
2. Activate the plugin through the ‘Plugins’ screen in WordPress.
3. Go to **Settings > Wishlist** to configure which post types to support.

== Frequently Asked Questions ==

= Does it support WooCommerce? =  
Yes, WooCommerce products are fully supported, including price and add-to-cart integration.

= Can I change the "Add to Wishlist" text? =  
Yes, you can customize the wishlist button text from the plugin settings.
